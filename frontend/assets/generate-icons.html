Imagify AI — App Icon Generator

Click an icon to download it. Save each file to frontend/assets/ with the suggested filename.

What each file does:
• icon.png — iOS app icon & general fallback (1024×1024 full-bleed).
• adaptive-icon.png — Android adaptive foreground. The brand mark sits inside Android's 66% safe zone (center 680×680 of 1024×1024) so launchers can mask it as a circle/rounded-square without clipping. The solid color behind is set in app.json.
• favicon.png — web favicon, 48×48.
• splash-icon.png — transparent 512×512 mark used by expo-splash-screen (optional).
